<h2>Σύνοψη</h2>
Για την επλήρωση των παραδοτέων, επέλεξα συγκεκριμένες ασκήσεις προγραμματισμού και γραμμών εντολών. Για τις ασκήσεις προγραμματισμού επέλεξα τις: Zoom Hover Effect, Form Validation, Menu Scroll και Mouse Eraser. Αυτές οι ασκήσεις παραδόθηκαν όλες εντός των ορισμένων προθεσμιών τους. Δεν χρειάστηκε να κάνω περαιτέρω αναζητήσεις για να ολοκληρώσω τα ζητούμενα των ασκήσεων εκτός μονο μιας, της Menu Scroll, όπου χρειάστηκε να ανατρέξω σε μια ιστοσελίδα για να βρώ βοήθεια για κάποιες γραμμές κώδικα. Ο τρόπος με τον οποίο διαχειρίστηκα τις ασκήσεις ήταν κυρίως με πειραματισμό, μελετώντας τον υπάρχων κώδικα και γράφοντας τον δικό μου ανάλογα. Για τις ασκήσεις γραμμών εντολών επέλεξα τις εξής: Taskwarrior, vimrc, youtube dl και mpv player και emacs editor. Αυτές οι ασκήσεις παραδόθηκαν όλες εντός των ορισμένων προθεσμιών τους, πλήν της τελευταίας η οποία παραδόθηκε μια εβδομάδα αργότερα λόγω τεχνικών προβλημάτων που αντιμετώπιζα με το virtual machine μου που χρησιμοποιώ για να προσομοιώσω σύστημα Linux. Για την εκπλήρωση αυτών των ασκήσεων έκανα αναζητήσεις στο διαδίκτυο για να κατανοήσω την χρησιμότητά των προγραμμάτων και το πως λειτουργούν. Υπάρχουν σχετικές αναφορές στις πηγές που συμβουλεύθηκα στο τέλος του κάθε παραδοτέου. Για την δημιουργία των εικόνων gif χρησιμοποίησα το πρόγραμμα OBS για την καταγραφή της οθόνης μου και την ιστοσελίδα <a href="https://ezgif.com/" title="ezgif site">ezgif.com</a> για την μετατροπή των καταγραφών μου σε εικόνες gif. Όλες οι εικόνες που εμφανίζονται εδώ, είναι αποθηκευμένες σε προσωπικό μου αποθετήριο.

<h2>Συμπεράσματα</h2>
Κοιτόντας πίσω στον χρόνο και πιο συγκεκριμένα στις αρχές του εξαμήνου, πριν ξεκινήσω να ασχολούμαι με το μάθημα, η διαφορά στις γνώσεις μου για τα αντικείμενα ειναι σημαντική. Δουλεύοντας κάθε εβδομάδα σε ενα διαφορετικό παραδοτέο, με βοήθησε να εξοικειωθώ με ορισμένες τεχνικές προγραμματισμού, κυρίως μέσω των ασκήσεων προγραμματισμού. Οι ασκήσεις γραμμής εντολών μου έδωσαν να καταλάβω πόσο απαιτητική μπορεί να γίνει η εργασία μέσω terminal αποκλειστικά, καθώς σε αυτές αντιμετώπισα τις περισσότερες δυσκολίες. Η παρακολούθηση των εργαστηρίων του μαθήματος αποδείχθηκε ιδιαίτερα σημαντική διότι μου λήθηκαν αρκετές απορίες. Οι δυσκολίες που συνάντησα με τα submodules μου εδείξε πόσο προσεκτικά πρέπει να ακολουθώ τα βήματα για διαδικασίες του git. Η αδυναμία να καταφέρω να κάνω τα submodules να λειτουργήσουν σωστά μου δημιούργησε πρόβλημα με την παράδοση και των δυο μερών του συμμετοχικού περιεχομένου, τα οποία παρέδωσα και τα δυο με λίγες μέρες διαφορά απο την διωρία τους.
