# **Εισαγωγή**
## **Οι Προσδοκίες μου από το Μάθημα**
Μέχρι στιγμής από το παρόν μάθημα περιμένω να αποκτήσω μια ποικιλία γνώσεων που θα μου χριαστούν στο μέλλον.
Αρχικά, με την εξοικείωσή μου με το GitHUb θα εχω την δινατότητα να το χρησιμοποιήσω με άνεση στο μέλλον όχι μόνο στα υπόλοιπα μαθήματα αλλά και για προσωπική χρήση.
Επιπλέον, η εγκατάσταση Archlinux και η χρήση του θα με βοηθήσουν να μάθω να χρησιμοποιώ παρόμοια συστήματα διάδρασης, καθώς και θα επεκτείνουν τις προγραμματιστικές μου γνώσεις. Τέλος η άμεση χρήση διαφόρων ειδών διεπαφών θα με βοηθίσει στην πιο πλήρη κατανόησή τους. <br />
# **Άσκηση γραμμής εντολών - 2ο παραδοτέο**
Αρχικά επέλεξα την άσκηση γραμμής εντολών που ζητάει να ελέγξω τον καιρό από δύο πόλεις στα warm-up, καθώς είναι πολύ απλό το installation και θα με βοηθήσει να εξοικειωθώ με την λήψη και εκτέλεση packages. Αυτό γίνεται με την χρήση του ``curl`` package που δίνει πληροφορίες για διάφορες ιστοσελίδες και του ``wttr.in/`` πού λαμβάνει πληροφορίες για τον καιρό. Η καταγραφή έγινε σε terminal μέσω ``asciicast``.

# **Άσκηση γραμμής εντολών - 3ο παραδοτέο**
Στην 2η άσκηση γραμμής εντολών διάλεξα και πάλι απο τα warm up όπυ μας ζητάει να δείξουμε πληροφορίες συστήματος και τα dot αρχεία. Αυτό το έκανα με την χρήση των πακέτων  ``neofetch``  και  ``ls``. Η καταγραφή του terminal έγινε και πάλι με ``asciicast``.
