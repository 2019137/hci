# ΕΙΣΑΓΩΓΗ
  Σκοπός του μαθήματος είναι η εξικοίωση του φοιτητή με τα user-interfaces,τόσο στο σχεδιασμό τους όσο και στις τεχνικές αλληλεπίδρασης.Ο δικός μου στόχος(σκοπός)είναι να μπορέσω να ανταποκριθώ στις απαιτήσεις του μαθήματος αλλά και στο τέλος του εξαμήνου να είμαι σε θέση να αναγνωρίζω τεχνικές και μεθόδους για την ευχρηστία συνεργατικών (και μή) συστημάτων.Μέσα από μελέτη και παρακολούθηση τόσο των θεωρητικών διαλέξεων όσο και τών εργαστηριακών,καθώς και με τη συμμετοχή και την έγκαιρη αποστολή των εβδομαδιαίων παραδοτέων,πιστεύω ότι τελικά θα καταφέρω να φτάσω στο στόχο μου.Πολλές έννοιες,απο την πρώτη κι'όλας διάλεξη,μου ήταν άγνωστες(π.χ markdown κώδικας)αλλά μέσα από την βιβλιογραφία που υπάρχει στο μάθημα,μπόρεσα να τις μάθω.Το μάθημα είναι ιδιαίτερα σημαντικό για μένα και πολύ ενδιαφέρων για το λόγο ότι η επιλογή κατεύθυνσης μου θα είναι Ανθρωπιστικές και Κοινωνικές Επιστήμες,επομένως θα προσπαθήσω να αποκομίσω όσων των δυνατών περισσότερα απο το μάθημα.
