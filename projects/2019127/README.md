|1ο Παραδοτέο|
Στο πρώτο παραδοτέο κάναμε fork το hci και στην συνέχεια δημιουργήσαμε ένα καινούργιο file όπου γράψαμε το ονοματεπώνυμο,ΑΜ και αντιγράψαμε τον παραπάνω πίνακα.
|Πρώτη επαφή|
Αρχικά,φάνηκε αρκετά περίπλοκο εφόσον δεν έχουμε εξοικειωθεί σε αυτό το περιβάλλον,ωστόσο με την βοήθεια των καθηγητών επιλύθηκαν οι απορίες.
|Ανάγκες|
Μια από τις κυριότερες ανάγκες είναι να εξοικειωθούμε στο github και LInux/Unix μιας και θα τα χρησιμοποιούμε καθ'ολη την διάρκεια του εξαμηνου.
|Στόχους|
Ένας από τους στόχους μου για το μάθημα είναι να αξιοποιήσω τις γνώσεις και την εμπειρία που θα αποκτήσω από αυτό.Ακόμα ένας στόχος είναι να κατανοήσω περισσότερο μέσω του μαθήματος τη φύση του υπολογιστή.
