<h2><span id="Προσδοκίες από το μάθημα">(1) Προσδοκίες από το μάθημα</span></h2>

<h2><span id="Image Processing">(2) Image Processing (Άσκηση Προγραμματισμού)</span></h2>
<p>Η άσκηση που επέλεξα είχε να κάνει με την επεξεργασία των φίλτρων μίας εικόνας, συνδυασμό αυτών και ρύθμιση φωτεινότητας, αντίθεσης κλπ. Δημιούργησα τρία παραδείγματα της εικόνας με φίλτρα, το καθένα με διαφορετικά χαρακτηριστικά για να παρατηρήσει κανείς πόσο μπορεί να αλλάξει η μορφή μιας εικόνας με λίγες γραμμές εντολών. Υλοποιήθηκε σε HTML και CSS και χρησιμοποίησα τα σχόλια που υπήρχαν στο τμήμα της CSS για την προσθήκη νέων φίλτρων στην εικόνα. Για πρώτη άσκηση με δυσκόλεψε λίγο, αλλά τελικά κατάφερα να την εκτελέσω και να ολοκληρώσω το ζητούμενο.

<h2><span id="Check the Weather">(5) Check the Weather (Άσκηση Γραμμής Εντολών)</span></h2>
<p>Η δεύτερη άσκηση γραμμής εντολών που υλοποίησα είχε να κάνει με τον έλεγχο του καιρού σε όποια περιοχή επιθυμούσα. <h2><span id="Download MP3">(10) Download MP3 (Άσκηση Γραμμής Εντολών)</span></h2>
<p>Η τελευταία άσκηση γραμμής εντολών, είχε να κάνει με την εγκατάσταση αρχείου .mp3 από το youtube downloader. Αρχικά, έπρεπε να κατεβάσω το ίδιο το Youtube Downloader με την χρήση της εντολής "sudo wget *link YD* -O /usr/local/bin/youtube-dl" και "sudo chmod a+rx /usr/local/bin/youtube-dl" (χωρις τα εισαγωγικά βέβαια). Μετά την ολοκλήρωση αυτής της διαδικασίας, με την εντολή "youtube-dl -x--audio-format mp3 *link of the song*" κατέβασα το τραγούδι που ήθελα. Επιπλέον, ωστέ να παίξει το κομμάτι, κατέβασα ένα audio player με την εντολή "sudo apt-get install sox libsox-fmt-all" και χρησιμοποίησα την εντολή "play 'name of the song' ". Ωστόσο, για κάποιο λόγο που δεν κατάλαβα, το τραγούδι δεν έπαιζε (σε όσα και να το δοκίμασα).

<h2><span id="Τελική Αναφορά">(12) Τελική Αναφορά</span></h2>
Ύστερα από πολλή δουλειά, κατάφερα να φτάσω στο τελευταίο παραδοτέο, νιώθοντας ότι κάθε στόχος μου εκπληρώθηκε. Εκτός αυτού όμως, δημιουργήθηκαν περισσότεροι προσωπικοί στόχοι. 
<br><br>Χάρη στις εβδομαδιαίες ασκήσεις προγραμματισμού και γραμμής εντολών πλέον νιώθω ότι έχω αναπτύξει μεγάλες δεξιότητες πάνω στον προγραμματισμό, αλλά προπάντων την χρήση του τερματικού. Αντίστοιχα, νιώθω πολύ εξοικειωμένος με την χρήση του GitHub, μια πλατφόρμα που ο καθηγητής μας γνώρισε, μαζί με άλλες χρήσιμες (π.χ. το Codepen) και εργαλεία που χρησιμοποιήσαμε (π.χ. το Asciinema) και θα συναντήσουμε μελλοντικά στο πανεπιστήμιο ή δική μας χρήση. Επιπροσθέτως, έμαθα να κάνω πιο ακριβή αναζήτηση στο διαδίκτυο ώστε να βοηθηθώ με τις απορίες σε προβλήματα που προέκυπταν.
<br><br>Αξιοσημείωτο το γεγονός ότι δημιουργήθηκαν περισσότεροι προσωπικοί στόχοι. Ενας σημαντικός στόχος για μένα πλέον είναι η μεγαλύτερη εξοικείωση μου με το τερματικό και η άριστη χρήση του, καθώς αποτελεί απαραίτητο εργαλείο ενός προγραμματιστή. Επίσης, θέλω να ενισχύσω τις γνώσεις μου πάνω στις γλώσσες HTML, CSS και JS που χρησιμοποιήσαμε σε όλες τις ασκήσεις προγραμματισμού και θα φανούν χρήσιμες στο χτίσιμο ιστοσελίδων. Επιπλέον, θέλω να μάθω να χρησιμοποιώ το GitHub καλύτερα διότι θα μου φανεί πολύ χρήσιμο σε μελλοντικά projects.
<br><br>Η αλήθεια είναι ότι δυσκολεύτηκα με τις ασκήσεις γραμμής εντολών, καθώς πρώτη φορά δούλευα στο τερματικό του υπολογιστή μου. Οι ασκήσεις προγραμματισμού ήταν πιο εύκολες, όλες είχαν πολύ ενδιαφέρον και το γεγονός ότι βλέπαμε σε εικόνα το αν κάναμε κάτι σωστά ή λάθος ήταν αυτό που τις καθιστούσε ιδιαίτερες. Το συμμετοχικό περιεχόμενο ήταν κάτι διαφορετικό σε σχέση με τις παραπάνω ασκήσεις καθώς περιλάμβανε έρευνα και συνεισφορά στο βιβλίο του καθηγητή.
<br><br>Τέλος, νιώθω πολύ ευχαριστημένος που δούλεψα σκληρά για κάθε μία άσκηση και κατάφερα να εκπληρώσω όλους τους στόχους μου.
