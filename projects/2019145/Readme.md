# **Εισαγωγή** 

Στόχος μου στα πλαίσια του μαθήματος είναι να αποκτήσω την ικανότητα να δημιουργώ user interfaces που να μεγιστοποιεί την άνεση του χρήστη, καθώς και να είναι αποτελεσματικό με τέτοιο τρόπο ώστε ο χρήστης να μπορεί να εκπληρώσει οτιδήποτε επιθυμεί. Επιπλέον, θα είναι χρήσιμο να γνωρίζω τους τρόπους με τους οποίους θα κατανοώ τι χρειάζεται ένας χρήστης από ένα user interface, έτσι ώστε να μπορώ να προσφέρω το καλύτερο δυνατό αποτέλεσμα. Ως προς το πρόγραμμα του μαθήματος, δεν έχω καμία ιδιαίτερη ανάγκη. Για να επιδιώξω τους στόχους μου, θα παρακολουθώ τα μαθήματα έτσι ωστέ να αποκτήσω όσες πιο πολλές πληροφορίες και γνώσεις μπορώ, και στον δικό μου χρόνο θα πραγματοποιώ τις εργασίες που χρειάζέται να παραδώσω με συνέπεια.

# **Σύνοψη**

Όλα τα παραδοτέα που επέλεξα έχουν υλοποιηθεί πλήρως. Σχετικά με τις ασκήσεις γραμμής εντολών, αρκέστηκα στα κάθε ζητούμενα της άσκησης και δεν επεκτάθηκα παραπάνω. Με τις ασκήσεις προγραμματισμού δεν υπήρξε ιδιαίτερη δυσκολία καθώς γνώριζα ήδη τα βασικά της HTML και της CSS. Για τις ασκήσεις γραμμής εντολών, δυσκολεύτηκα αρκετά σε μερικά σημεία καθώς δεν είχα χρησιμοποιήσει shell σε τέτοια κλίμακα. Ωστόσο, τώρα είμαι πιο οικείος με την χρήση του. Σε κάθε asciinema link, εξηγώ και τι κάνω στο terminal. Στο συμμετοχικό περιεχόμενο, αναφέρθηκα στο τηλεχειριστήριο του NES και στο Wii remote καθώς και το πως αυτά τα δύο άλλαξαν την διάδραση χρήστη-βιντεοπαιχνιδιών. Επιπλέον, εξέτασα την περίπτωση του Nintendo Switch και το πως έφερε έναν τελείως καινούριο και μοναδικό τρόπο για έναν χρήστη να χρησιμοποιήσει μια κονσόλα. Τέλος, αναφέρθηκα στο Shigeru Miyamoto λόγω της τεράστιας συνεισφοράς του στον χώρο των βιντεοπαιχνιδιών.

# Συμπεράσματα

Σχετικά με τις ανάγκες μου από της αρχή του εξαμήνου, θεωρώ ότι έχουν εκπληρωθεί σε ικανοποιητικό βαθμό. Έχω κατανοήσει τις έννοιες σχετικά με την διάδραση χρήστη-υπολογιστή και γνώριζω τις πρακτικές χρήσεις της. Παρόλο που η μορφή και η δομή του μαθήματος ήταν διαφορετική από αυτήν που περίμενα αρχικά, έμαθα να χρησιμοποιώ νέα εργαλειά, συγκεκριμένα to bash shell. Μέσω των εργασίων προγραμματισμού, έχω πλέον μεγαλύτερη κατανόηση των HTML, CCS και Javascript. Το συμμετοχικό περιεχόμενο μου έδωσε την δυνατότητα να αναπτυχθώ σε ένα θέμα που με ενδιαφέρει και να το συσχετίσω με την διάδραση του χρήστη. Βρισκόμουν στις διαλέξεις των μαθήματων και παρέδιδα τα παραδοτέα έγκαιρα, οπότε θεώρω σε αυτό το κομμάτι είμαι πλήρως καλυμμένος. Ήδη έχω χρησιμοποιήσει μερικά από τα πράγματα που έχω μάθει από το μάθημα σε προσωπικά μου project και ελπίζω να μπορώ να συνεχίσω να χρησιμοποιώ αυτήν την γνώση.
