<h2><span id="Τελική αναφορά">(12) Τελική αναφορά*</span></h2>
<li>Όταν για πρώτη φορά είδα το περιεχόμενο αυτού του μαθήματος στόχος μου έγινε να αποκομίσω όσες περισσότερες γνώσεις μπορώ. Πιο συγκεκριμένα, να μάθω πως να χειρίζομαι το github με τον καλύτερο δυνατό τρόπο (από τα βασικά μέχρι και κάποιες λίγο πιο εξειδικευμένες γνώσεις), κάτι που τελικά για να παραδώσει κάποιος την τελική αναφορά οφείλει να πετύχει. Ακόμη, να γνωρίσω και να εξοικειωθώ με γλώσσες όπως η html και η css (στις ασκήσεις προγραμματισμού χρησιμοποίησα αυτές τις δύο). Τέλος να μάθω να χειρίζομαι το terminal του λειτουργικού linux (ubuntu) ένα λειτουργικό που ο κάθε προγραμματιστής πρέπει να χειρίζεται με ευκολία και που σχεδόν όλες οι διεργασίες πραγματοποιούνται μέσω του terminal του λειτουργικού.<br>
<li>Από τις ασκήσεις προγραμματισμού γνώρισα κάποια βασικά commands στις css και html που θα μου φανούν αρκετά χρήσιμα, εξαιτίας του πόσο διαδεδομένες και χρήσιμες αυτές οι γλώσσες είναι. Επίσης ανακάλυψα τρόπους επεξεργασίας μίας εικόνας ακόμα και αν είναι τύπου svg με σκοπό την προσαρμογή και την αφομοίωση της σε π.χ. μία ιστοσελίδα καθώς και την δημιουργία διαφόρων 'effect' που ενεργοποιούνται από συγκεκριμένες κινήσεις του χρήστη για παράδειγμα ο χρήστης κινεί το ποντίκι. Επίσης μπορώ πλέον να δημιουργήσω φόρμες ερωτήσεων που χρησιμοποιούν το ποντίκι ως θέση εισόδου και θα ήθελα να δω τι παραπάνω μπορώ να κάνω (μία καλή ιδέα για μελλοντική εργασία μαθήματος αν προκύψει ή για προσωπική μελέτη). Άσχετα από τις προγραμματιστικές γνώσεις όμως, έμαθα να χειρίζομαι το terminal του ubuntu ξεκινώντας από βασικές εντολές γραμμής εντολών όπως cd και ls και συνεχίζοντας με sudo commands κ.α. για να κατεβάσω και να χρησιμοποιώ εργαλεία που θα μπορούσαν να φανούν χρήσιμα στο λειτουργικό ubuntu. Τέλος, κάτι άλλο που έμαθα είναι χρήσιμες πλατφόρμες και τρόπους αξιοποίησής τους όπως το github, το codepen, το asciinema και το netlify που έχουν αμέτρητες πληροφορίες και λειτουργείς που μπορεί να αξιοποιήσει ένας προγραμματιστής. Γνωρίζοντας έτσι, την πιο  εξειδικευμένη χρήση μίας πλατφόρμας και απέκτησα βάσεις οι οποίες θα φανούν χρήσιμες, σε οποιονδήποτε τομέα του προγραμματισμού και να ακολουθήσω στο μέλλον.<br>
<li>Αυτές οι βάσεις λοιπόν, ήταν κάτι που δεν είχα σκεφτεί ότι μπορώ να κερδίσω και απορώ διότι τις θεωρώ σημαντικότερες από τους στόχους μου στην αρχή του μαθήματος καθώς πλέον μπορώ να αποκτήσω γνώσεις χωρίς την βοήθεια κάποιου, δηλαδή μου ανοίγεται μια νέα πόρτα, με γνώσεις της δικής μου επιλογής. Αυτός είναι και ο λόγος που έμεινα ευχαριστημένος από το μάθημα αυτό αφού ξεπέρασε τις προσδοκίες μου για γνώση που ήταν ο αρχικός μου στόχος αλλά και και κάτι πιο σημαντικό την προσωπική καθοδήγηση, ένα εφόδιο που θα με βοηθήσει να βελτιώνομαι για μια ζωή.
