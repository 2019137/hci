Οι στόχοι-ανάγκες που έχω για το συγκεκριμένο μάθημα είναι να έρθω όσο πιο κοντά μπορώ με τον υπολογιστή ,να κατανοήσω την εξέλιξη των υπολογιστών στις σύχγρονες μέρες που διανύουμε,όσο αναφορά την επικοινωνία του ανθρώπινου είδους με την υπολογιστική μηχανή , και γενικότερα να χρησιμοποιήσω διαφορετικά εργαλεία πληροφόρησης και υπολογισμού ,ωστε να διερύνω τους ''πνευματικούς'' μου ορίζοντες.
